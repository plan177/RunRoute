# ТЗ: Редизайн UI RunRouteBot Mini App

## 1. Общая концепция

Мини-приложение для Telegram, вдохновлённое стилем приложения **BEGY** (беговое сообщество). Тёмная тема, минимализм, акцент на зелёном. Мобильный-first дизайн.

---

## 2. Цветовая палитра

| Элемент | HEX | Описание |
|---|---|---|
| Фон основной | `#1B1B2F` | Тёмно-синий/чёрный |
| Фон карточек | `#232340` | Слегка светлее фона |
| Акцент основной | `#8BC34A` | Лайм-зелёный (как в BEGY) |
| Акцент hover | `#9CCC65` | Светлее при наведении |

## 4. Структура экранов

### 4.1 Сplash Screen (загрузка)
- Фон: градиент `#1B1B2F` → `#2D4A22` (тёмный к зеленоватому)
- Логотип по центру: иконка бегуна + название "RunRoute"
- Текст: "твоё пространство для бега"
- Прогресс-бар: зелёный `#8BC34A`
- Время показа: 2-3 секунды

### 4.2 Экран "Главная" (Лента)
**Шапка:**
- Приветствие: "Добро пожаловать, {имя}!"
- Аватар пользователя (круглый, 40px)
- Кнопка "Закрыть" (крестик) — сверху справа

**Секция "Мои события":**
- Заголовок: "События не запланированы"
- Ссылка: "Запланировать" (зелёный текст)

**Секция "Лента новостей":**
- Карточки постов:
  - Аватар автора + Имя + Время
  - Текст поста
  - Изображение (если есть)
  - Кнопки: лайк, комментарий, поделиться

### 4.3 Экран "Маршруты" (замена "Авто/Рисование/Текст")

**Шапка:**
- Заголовок: "Мои маршруты"
- Кнопка "+" (добавить новый)

**Фильтры:**
- Горизонтальный скролл: "Все", "5 км", "10 км", "Полумарафон", "Марафон"

**Карточки маршрутов:**
- Название маршрута
- Дистанция + Время
- Маленькая карта-превью
- Дата создания

**Кнопка "Построить маршрут"** — фиксированная внизу, зелёная

### 4.4 Экран "Календарь"

**Шапка:**
- Стрелки влево/вправо для навигации по месяцам
- Текущий месяц и год (centered)

**Сетка календаря:**
- Пн, Вт, Ср, Чт, Пт, Сб, Вс
- Дни с событиями — подсвечены зелёным кружком
- Текущий день — зелёный контур
- Выбранный день — заливка зелёным

**Кнопка внизу:**
- "Добавить запись" — зелёная, полная ширина

### 4.5 Экран "Профиль"

**Верхняя часть:**
- Аватар (круглый, 80px)
- Имя пользователя
- Статистика: маршруты / км / время

**Настройки:**
- Список опций с иконками
- Переключатель темы (тёмная/светлая)
- Выход

## 10. Расчёт темпа бега

### 10.1 Формула
```
темп (мин/км) = время (мин) / дистанция (км)
```
Пример: 30 мин / 5 км = 6:00 мин/км

### 10.2 Отображение на карточке маршрута
```
┌─────────────────────────────┐
│ 🏃 Маршрут "Парк Горького"  │
│                             │
│  5.2 км    31 мин    6:00   │
│  дистанция  время   темп/км │
│                             │
│  [карта-превью]             │
└─────────────────────────────┘
```

### 10.3 Отображение после построения маршрута
Блок "Статистика маршрута" под картой:
```
┌────────────────────────────────────┐
│  📏 5.2 км  │  ⏱ 31 мин  │  🏃 6:00  │
│  дистанция   │  время      │  темп/км  │
└────────────────────────────────────┘
```

### 10.4 Цвета по темпу
| Темп | Цвет | Описание |
|---|---|---|
| < 4:30 | `#EF5350` (красный) | Быстрый / спринт |
| 4:30 - 5:30 | `#FF9800` (оранжевый) | Интенсивный |
| 5:30 - 6:30 | `#8BC34A` (зелёный) | Комфортный |
| 6:30 - 8:00 | `#29B6F6` (голубой) | Лёгкий / разминка |
| > 8:00 | `#8E8EA0` (серый) | Прогулка |

### 10.5 Интерактив
- При выборе дистанции показывать **ожидаемый темп** (6:00 мин/км по умолчанию)
- После построения маршрута — **реальный расчётный темп**
- В календарии — средний темп за месяц
- На экране профиля — средний темп за всё время

### 10.6 Формат вывода
```
мин:сек мин/км
```
Примеры: `5:42 мин/км`, `6:00 мин/км`, `7:15 мин/км`
